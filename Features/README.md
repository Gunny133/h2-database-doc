## 特性列表

### 主要特性

* 非常快的数据库引擎
* 开源
* Java 编写
* 支持标准 SQL, JDBC API
* 内嵌和服务器模式，支持集群
* 强大的安全特性
* 可使用 PostgreSQL ODBC
* 多版本并发

### 其他特性

* 基于磁盘或者内存数据库和表，支持只读数据库，临时表
* 支持事务（read committed）,两阶段提交
* 多连接，表级锁
